# ストア掲載アセット

Apple App Store と Google Play 向けの、ローカライズされたストア掲載コンテンツです。

## ディレクトリ構成

```
store-listing/
  android/
    <locale>/
      title.txt              # アプリタイトル（最大 30 文字）
      short-description.txt  # 短い説明（最大 80 文字）
      full-description.txt   # 詳しい説明（最大 4000 文字）
  ios/
    <locale>/
      description.txt        # App Store の説明
```
