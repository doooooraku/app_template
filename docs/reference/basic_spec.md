**重要ルール**:

- ファイルパス（写真 / 音声等）は **必ず相対パス**で保存する。`src/db/filePathUtils.ts` を経由すること。理由: iOS Store 更新でコンテナ UUID が変わると絶対パスは破損する。

### 6.2 アクセシビリティ（最低ライン）

- タップ領域は 44x44pt 以上
- 重要文言はアイコンだけにせずラベル併記
- ダークモード対応（system / light / dark）
- WCAG AA レベルのコントラスト比（4.5:1 以上）

### 6.3 セキュリティ / プライバシー

- 個人情報（氏名 / メール / 電話 / 住所）をアプリ要件として取得しない
- 課金情報はストアが管理し、アプリは保持しない
- API キー / シークレットは `.env` から読み込み、コードに直書きしない
- ログに PII を出さない

詳細: [`constraints.md`](./constraints.md)

## 7. 多言語対応

- 対応言語: 19 言語（en, ja, fr, es, de, it, pt, ru, zh-Hans, zh-Hant, ko, hi, id, th, vi, tr, nl, pl, sv）
- レイアウトは LTR のみ（RTL は v1 スコープ外）
- 全ての user-visible な文字列は `src/core/i18n/` の翻訳キーを経由する
- 翻訳キー追加時は **全 19 言語ファイルに明示的に追加**する（baseEn フォールバックに頼らない）
